# Advent of Code 2025

My Zig project in which I try to solve [Advent of Code 2025](https://adventofcode.com/2025/) puzzles.

## Inputs

Before running remember to put your inputs in the [inputs](inputs/) folder.

## Usage

To run a solution for a selected day, edit [main.zig](src/main.zig) file accordingly and run:

```console
foo@bar:~$ zig build run
```
